# srv1719193

Public Hostinger VPS. Receives **outbound** reverse SSH tunnels from ms02 so
the operator can reach the dev host while away from the home LAN.

| Field | Value |
|---|---|
| FQDN | `srv1719193.hstgr.cloud` |
| Public IP | `76.13.1.95` |
| Ansible group | `tunnel_hubs` |
| Ansible user | `root` |
| Playbook | `playbooks/ms02_reverse_tunnel.yml` |
| ms02 reverse port | **22002** → `127.0.0.1:22` on ms02 |

## sshd

`GatewayPorts clientspecified` (drop-in
`/etc/ssh/sshd_config.d/99-reverse-tunnel.conf`) so ms02 can bind
`-R 0.0.0.0:22002:…`.

## authorized_keys

Managed idempotently by `roles/tunnel_hub` — ms02 `casibbald` + Mac
`picolino` Ed25519 keys (see `inventory/group_vars/tunnel_hubs.yml`).

## Reach ms02 from away

```bash
ssh ms02-away                    # Mac alias
ssh -p 22002 casibbald@76.13.1.95
```
